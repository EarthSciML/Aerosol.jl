# ISORROPIA II

ISORROPIA II is a computationally efficient thermodynamic equilibrium model for K+-Ca2+-Mg2+-NH4+-Na+-SO42--NO3--Cl--H2O aerosol systems. The implementation is a work in progress.

**Reference**: Fountoukis, C. and Nenes, A., 2007. ISORROPIA II: a computationally efficient thermodynamic equilibrium model for K+-Ca2+-Mg2+-NH4+-Na+-SO42--NO3--Cl--H2O aerosols. *Atmospheric Chemistry and Physics*, 7(17), pp.4639-4659. [https://doi.org/10.5194/acp-7-4639-2007](https://doi.org/10.5194/acp-7-4639-2007)
